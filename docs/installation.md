# Installation

Ubunye Engine supports Python 3.9 – 3.11.

---

## 🔧 Basic install
```bash
pip install ubunye-engine
```
________
## With Spark
```bash
pip install ubunye-engine[spark]
```

## With ML backends
```bash
pip install ubunye-engine[ml]
```

## Dev install
```bash
git clone https://github.com/your-org/ubunye-engine.git
cd ubunye-engine
pip install -e .[dev,spark,ml]
```
