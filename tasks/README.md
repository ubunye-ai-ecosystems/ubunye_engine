# tasks/

Scratchpad for work in flight. Not authoritative — GitHub issues /
changelog are.

- `done/` — finished work, newest suffix wins.
- `todo/` — queued work. `todo/task-00.md` is the umbrella strategy;
  the rest are concrete pieces of it.

When a todo flips to done, move the file (don't copy), keep its number,
and append a **Status:** done (date) line at the top.
